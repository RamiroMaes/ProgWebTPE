Al agregar un jugador via formulario se muestra al final de la lista cuando en la base de datos se inserta donde corresponde.
Para que ver el jugador en la posición correcta hay que recargar la página completa.

Posible solución:
En el formulario (entify_form.templ): Cambia el hx-target para que apunte a la tabla completa y el hx-swap para reemplazarla (outerHTML).
// ...existing code...
templ EntityForm (paises []string){
    <form
        hx-post="/jugadores"
        hx-target="#tabla-jugadores"
        hx-swap="outerHTML"
        hx-on:htmx:response-error="alert(event.detail.xhr.responseText)"
        hx-on:htmx:after-request="if(event.detail.successful) this.reset()"
    >
        Posición (seleccione una):
// ...existing code...


En el handler (jugadores.go): En lugar de devolver solo la fila (EntityRow), recupera la lista actualizada (ListPlantel) y devuelve la tabla completa (EntityList).
// ...existing code...
        queries := db.New(dbConn)
        
        // 1. Insertamos el jugador en la base de datos
        _, err := queries.CreateJugador(r.Context(), params)
        if err != nil {
            if pqErr, ok := err.(*pq.Error); ok {
                // Manejo de errores de Postgres (duplicados, constraints, etc)
                http.Error(w, "Error de base de datos: "+pqErr.Message, http.StatusBadRequest)
            } else {
                http.Error(w, "Error interno al crear el jugador: "+err.Error(), http.StatusInternalServerError)
            }
            return 
        }

        // 2. CAMBIO IMPORTANTE: 
        // Recuperamos el plantel completo nuevamente. 
        // La base de datos se encarga de aplicar el ORDER BY (Arquero, Defensor, etc.)
        plantel, err := queries.ListPlantel(r.Context())
        if err != nil {
            http.Error(w, "Error al actualizar la lista: "+err.Error(), http.StatusInternalServerError)
            return
        }

        // 3. CAMBIO IMPORTANTE:
        // Renderizamos el componente de la TABLA completa (EntityList), no solo la fila.
        // Esto reemplazará la tabla vieja con la nueva ordenada.
        w.Header().Set("Content-Type", "text/html")
        views.EntityList(plantel).Render(r.Context(), w)
    }
}

Basicamente en vez de renderizar solo la fila nueva, se renderiza toda la tabla actualizada haciendo un llamado al query que devuelve la lista ordenada.
